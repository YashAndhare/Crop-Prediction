# 🌱 Crop Prediction Model using Machine Learning

## 📌 Overview

This project is a Machine Learning-based web application that predicts the most suitable crop based on soil nutrients and environmental conditions. It helps in making smart agricultural decisions using data-driven insights

---

## 🚀 Live Demo

👉 https://crop-prediction-ec9mrce52andkmy8bjoygm.streamlit.app/

---

## 🧠 Features

* Predicts crop based on:

  * Nitrogen (N)
  * Phosphorus (P)
  * Potassium (K)
  * Temperature
  * Humidity
  * pH
  * Rainfall
* User-friendly web interface using Streamlit
* Fast and accurate predictions using Random Forest
* Real-time deployment

---

## 🛠️ Tech Stack

* **Programming Language:** Python
* **Libraries:** NumPy, Pandas, Scikit-learn
* **Machine Learning Model:** Random Forest
* **Frontend:** Streamlit
* **Deployment:** AWS EC2 & Streamlit Cloud

---

## ⚙️ How It Works

1. User inputs soil and environmental parameters
2. Data is processed and passed to trained ML model
3. Model predicts the most suitable crop
4. Result is displayed instantly on the web app

## 📊 Project Workflow
